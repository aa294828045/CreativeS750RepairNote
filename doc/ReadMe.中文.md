### S750导电胶老化引起电源板击穿损坏
创新一直使用了一种电子黄胶-UL 1603，这种黄胶在一般温度下不会有事，但在在S750这种高温烧烤模式下，
在经过一定的时间，这种胶水就变成了棕色的、易碎的、导电的物质（正常的电子胶水都是绝缘的），加上电源板上最高电压300V，
有的地方的线路又靠的很近，高电压+导电物质=通电的一瞬间‘啪’的一声，完了!这就是为什么音箱在使用几年后普遍性的出故障~~~~~~
这些导电胶老化后会腐蚀元件以及电路板
